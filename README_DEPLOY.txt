
# Nova Vybes — Free Host Deploy

Railway (recommended):
1) Push this folder to a GitHub repo.
2) railway.app → New Project → Deploy from GitHub → select repo.
3) Variables: TELEGRAM_BOT_TOKEN, CRYPTOBOT_API_TOKEN, SOL_RECIPIENT (+ optional).
4) Start Command: node index.js → Deploy.

Render:
1) render.com → New → Background Worker → link repo
2) Build: npm install | Start: node index.js
3) Add same environment variables.
